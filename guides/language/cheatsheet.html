本速查基于 Google 工程师 Mehmet Fidanboylu 撰写的一份内部文档。该文档用来帮助 Google 工程师记忆一些 Dart 通用的语法功能。

有关这些功能的交互式指南，参见 Dart Codelab 速查。

字面量的使用

‘插值 ${val}’

在字面量字符串中引入 val 的值。 等价于： ‘Substitution’ + val

<type>[ ]

创建一个 List<type> 类型的对象。 等价于：new List<type>();

const [1, 2, 3]

创建一个编译时 List 常量。

= { }

初始化一个 Map 。 等价于：new Map<>();

字段声明

var 一般 var (动态的)
final 同 var 但仅能单次赋值
const 常量

类型检查

as 类型转换
is 示例类型判断
is! 示例类型判断结果取反

链式方法调用

a..b = true..c = 5;

使用级联来链接方法和访问其他成员。 等价于：a.b = true; a.c = 5;

null 处理

b ??= val;

如果 b 为 null ，设置 b 的值为 val ， 否则，保持 b 的原值。

a = value ?? 0;

如果 value 的值为 null ，设置 a 为 0 。 否则，设置 a 为 value 。

a?.b

条件访问。 等价于：a == null ? null : a.b

函数实现

fn({bool bold: false, bool hidden: false})

为命名参数设置默认值。

int incr(int a) => a + 1;

可以简化单个返回语句。

异常处理

try {...}
on MyException {...}
catch (e) {...}
finally {...}

通过 on 来捕获一种类型的异常。 通过 catch 来捕获异常的实例。

构造函数实现

标准构造函数（Normal constructor）

Point(this.x, this.y);

工厂构造函数（Factory constructor）

factory Point(int x, int y) => ...;

构造函数不总是创建一个新的对象时，使用 factory 方式实现构造函数。

命名构造函数（Named constructor）

Point.fromJson(Map json) {
    x = json['x'];
    y = json['y'];
  }

代理构造函数（Delegating constructor）

Point.alongXAxis(num x) : this(x, 0);

常量构造函数（Const constructor）

const ImmutablePoint(this.x, this.y);

生成一个永远不会改变的对象。 这里所有的变量必须都是 final 。

初始化列表

Point.fromJson(Map jsonMap)
      : x = jsonMap['x'], y = jsonMap['y'];

设置的字段为 final 时，使用初始化程序列表会很便捷。
