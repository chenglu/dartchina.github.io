获取 Dart SDK

大纲
  • 安装 Dart SDK
    • 升级
    • 切换版本
  • 关于发布渠道和版本号

Dart SDK 包含开发 Web ，命令行，服务端应用所需要的库和命令行工具， 如果仅需要开发移动端应用，只要安装 Flutter 即可，不需要 Dart SDK 。

要了解 Dart 开发中的其他可用工具，请参阅 Dart 工具 。 要了解 SDK 中的内容，请参阅Dart SDK 概述。

Note: 注意：本网站的文档和示例基于 Dart SDK 2.3.2 版本。

安装 Dart SDK

如下介绍，可以使用包管理器轻松的安装更新 Dart SDK。也可以，从源代码构建 SDK 或者下载 SDK zip 包文件。

Dart 工具可能会向 Google 发送使用指标和崩溃报告。 下载 Dart SDK，即表示您已同意 Google 服务条款。 注意：Google 隐私权政策 介绍了在此服务中的数据被如何处理。

  • Windows
  • Linux
  • Mac

选择以下一种方式安装：

使用 Chocolatey 安装

使用 Chocolatey 执行以下指令安装稳定版 Dart SDK：

C:\> choco install dart-sdk

执行以下指令安装开发板：

C:\> choco install dart-sdk --pre

执行以下指令升级 Dart SDK （升级开发板添加 --pre 参数）：

C:\> choco upgrade dart-sdk

使用设置向导安装

另外，你可以使用社区提供的 Windows 安装包。 通过设置向导安装 稳定版 或者 开发版 的 Dart SDK 。

如果是在 Debian/Ubuntu 系统的 AMD64 (64-bit Intel) 机器上，可选择以下一种方式安装， 两个方式都可以在发布新版本时自动更新 SDK 。

使用 apt-get 安装

执行以下一次性设置：

$ sudo apt-get update
$ sudo apt-get install apt-transport-https
$ sudo sh -c 'curl https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
$ sudo sh -c 'curl https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'

然后安装 稳定版 Dart SDK ：

$ sudo apt-get update
$ sudo apt-get install dart

或者，安装 开发板 Dart SDK ， 执行一次性设置命令：

$ sudo sh -c 'curl https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_unstable.list > /etc/apt/sources.list.d/dart_unstable.list'
$ sudo apt-get update
$ sudo apt-get install dart

安装 Debian 包

以.deb包格式

另外，可以下载 Debian .deb 格式的 Dart SDK 安装包。

  • Stable channel
  • Dev channel

修改访问 Dart 库的环境变量（PATH）

SDK 安装完成后，添加 SDK 的 bin 目录到环境变量中。例如： 使用以下命令更改当前终端会话中的环境变量：

$ export PATH="$PATH:/usr/lib/dart/bin"

要更改新建终端会话的环境变量，使用命令：

$ echo 'export PATH="$PATH:/usr/lib/dart/bin"' >> ~/.profile

安装 homebrew，然后执行以下命令：

$ brew tap dart-lang/dart
$ brew install dart

添加 --devel 参数安装稳定版：

$ brew install dart --devel

升级

升级 Dart 到一个新的可用版本：

$ brew upgrade dart

如果当前有效版本为开发版，想要安装稳定版，执行：

$ brew unlink dart
$ brew install dart

如果当前有效版本为稳定版，要升级到开发版，执行：

$ brew upgrade dart --devel --force

切换版本

切换本地已安装 Dart 版本，执行 brew switch dart <version> 。例如：

$ brew switch dart 1.24.3
$ brew switch dart 2.1.0

如果无法确定本地已安装了哪些版本，可执行：

$ brew info dart

该指令在顶部会打印最新的稳定版和开发版，随后会列出本地已安装的版本。

关于发布渠道和版本号

Dart SDK 有两个发布渠道：

  • 稳定版 渠道： 稳定发行版，每六周更新一次； 当前版本 [calculating].
  • 开发版 渠道： 预览发行版，通常每周更新一次； 当前版本 [calculating].

警告： 为了让开发者能够及早了解新功能和修复程序，发行的开发版并未经过像稳定版那样的严格测试。

稳定版 通道发行的 Dart SDK 版本具有 1.24.3 或 2.1.0 的版本号描述。 版本号是由点分隔的整数组成，不会包含连接符或字符。

开发版 通道发行的 Dart SDK 版本（预览发行版）版本号中包含以连接符开始的其他字符。 例如：Dart 2 预览发行版的版本号 2.0.0-dev.69.5 是以 2.0.0-dev 开始，并附加数字版本。

更多内容, 参见 Dart 2 章节。
