本章列出了一些 Dart 开发者发布的非常流行的有用的包。 到 Pub 网站发现更多包及核心库。

常用包分为以下三组：

  • 通用包
  • 在 Dart 核心库上的扩展包 {#packages-that-correspond-to-sdk-libraries}
  • 专业包
    • Flutter 包
    • Web 包
    • 命令行和服务器包

通用包

以下包被广泛的应用于项目中。

包 描述 常用 API
archive 用于各种存档和压缩格式的编解码。 Archive, ArchiveFile, TarEncoder, TarDecoder, ZipEncoder, ZipDecoder
http 一组可以轻松使用 HTTP 资源的高级函数和类。 delete(), get(), post(), read()
intl 用于国际化和本地化实施，支持复数表示，性别，日期，数字解析以及双向文本。 Bidi, DateFormat, MicroMoney, TextDirection
json_serializable 简单易用的用于代码生成的包。更多内容，参见 JSON 支持。 @JsonSerializable
logging 为应用增加可配置日志。 LoggerHandler, Level, LogRecord
mockito 在测试中 Mock 对象的流行框架。在依赖注入测试中非常有用。配合 test 包使用。 Answering, Expectation, Verification
path 为不同类型路径访问提供的通用操作。更多内容，参见 Unboxing Packages: path。 absolute(), basename(), extension(), join(), normalize(), relative(), split()
quiver Dart 核心库便捷工具。便捷工具为一些库提供额外扩展，这些库包括：Async， Cache， Collection， Core， Iterables， Patterns， 和 testing。 CountdownTimer (quiver.async); MapCache (quiver.cache); MultiMap, TreeSet (quiver.collection); EnumerateIterable (quiver.iterables); center(), compareIgnoreCase(), isWhiteSpace() (quiver.strings)
shelf Dart 的 Web 服务器中间件。 Shelf 可以轻松创建和组合 Web 服务器和 Web 服务器的部分服务。 Cascade, Pipeline, Request, Response, Server
stack_trace 由 Dart 底层实现用于堆栈跟踪信息解析、查看、处理的方法。还提供了堆栈跟踪信息字符串格式化生成函数。字符串格式化后的表示比原生堆栈跟踪信息具有更高的可读性。有关更多内容，请参见 Unboxing Packages: stack_trace。 Trace.current(), Trace.format(), Trace.from()
stagehand Dart 项目生成器。 WebStorm 和 IntelliJ 在创建新应用程序时使用 Stagehand 模板，这些模板也可以在命令行中使用。 Generally used through an IDE or the stagehand command.
test 在 Dart 中标准编写和运行测试的方式。 expect(), group(), test()
yaml YAML 解析器。 loadYaml(), loadYamlStream()

在 Dart 核心库上的扩展包

以下包基于核心库构建， 添加功能，并补全缺少的功能：

包 描述 常用 API
async 基于 dart:async 的扩展，增加 Utility 类用于异步计算。更多内容，参见 Unboxing Packages: async part 1， part 2， 以及 part 3。 AsyncMemoizer, CancelableOperation, FutureGroup, LazyStream, Result, StreamCompleter, StreamGroup, StreamSplitter
collection 基于 dart:collection 的扩展，增加, 增加 Utility 类，简化集合操作。更多内容参见 Unboxing Packages: collection。 Equality, CanonicalizedMap, MapKeySet, MapValueSet, PriorityQueue, QueueList
convert 基于 dart:convert 的扩展，增加编解码器，用于在不同的数据表示之间进行转换。其中具有代表性的 percent encoding ，也被称为 URL encoding 。 HexDecoder, PercentDecoder
io 包含两个库 ansi 和 io ，用来简化文件，标准流及进程处理。通过 ansi 可以实现自定义终端输出。 io 库用于处理进程，标准输入，以及文件拷贝等。 copyPath(), isExecutable(), ExitCode, ProcessManager, sharedStdIn
