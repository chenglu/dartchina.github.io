Obsolete pub features

大纲

As of Dart 2, pub no longer supports pub build, pub serve, or transformers. They’re replaced by the build system, which includes the build_runner tool.

For information about building and serving apps in Dart 2, see the following:

  • Web development:
    • Setup for Angular development (v5)
    • Deployment (v5)
    • Web-specific build_runner documentation
  • General:
    • Build system
    • Build system documents, including getting started with build_runner

If you use Dart 1.x for web development, see the following:

  • Setup for Angular development (v4)
  • Deployment (v4)

If you maintain a transformer, see the following:

  • Assets and Transformers in the archived Dart site
  • Writing a Pub Transformer in the archived Dart site

For help in switching from Dart 1.x to Dart 2, see the Dart 2 migration guides:

  • Language and core library migration guide
  • Web app migration guide
  • Barback/transformer migration guide
